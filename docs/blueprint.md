# **App Name**: SyncTube

## Core Features:

- Room Creation: Generate a unique, private room ID for synchronized listening.
- YouTube Playback Sync: Synchronize YouTube music playback across all participants with host priority.
- Drift Correction: Automatically correct playback drift every few seconds to maintain sync.
- Late Join Sync: Synchronize new participants with the current track and host timestamp.
- Audio/Video Toggle: Allow host to switch between audio-first and video-visible modes.
- Chat and Reactions: Enable text chat and emoji reactions during playback.
- YouTube Validation Tool: Use the YouTube Data API as a tool to validate video embeddability, excluding restricted content from use.

## Style Guidelines:

- Primary color: HSL 210, 60%, 50% -> RGB #3385FF - A vibrant blue that conveys a sense of modernity, connectivity, and energy, aligning with the app's real-time synchronization functionality.
- Background: Live changing ambient-type background design with bright colors that subtly shift and blend, creating a visually engaging and dynamic experience.
- Accent color: HSL 180, 50%, 50% -> RGB #40BFBF - A contrasting turquoise that adds a lively touch, distinct from the primary, highlighting interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral look, and excellent readability.
- Use clean, minimalist icons for playback controls and room settings, optimized for touch interactions.
- Mobile-first responsive design ensuring a seamless experience across various screen sizes, mimicking a native app interface.
- Incorporate subtle animations for feedback on user interactions, enhancing the app's responsiveness and user experience.
